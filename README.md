```markdown
# Handwashing M&E System for Health Setups

## Project Overview
This project focuses on monitoring and evaluating hand hygiene compliance within healthcare environments. By analyzing observational data, the system identifies gaps in hygiene practices to drive behavioral changes and improve patient safety.

## Key Features & Metrics
* **Overall Compliance Rate:** 64.75%.
* **Correct Technique Rate:** 63.29%.
* **Trend Analysis:** Tracked compliance growth from 38.10% in October to 81.97% by December.
* **Indications Analysis:** Identified that "Before a clean/aseptic procedure" is the most frequently missed indication (52.63%).
* **Departmental Insights:** Intensive analysis of the ICU/CCU showing a 65.29% compliance rate.

## Tools Used
* **SQL:** Data collection and cleaning of 800+ survey responses.
* **MS Power BI:** For interactive dashboarding and KPI tracking (12 KPIs total).

## Business Impact
* **Efficiency:** Reduced reporting time by over 90% (from 3 days to 2 hours).
* **Compliance:** Identified patterns that correlate with a 40% higher compliance rate.
* **Policy Influence:** Recommendations from this project were adopted by local health authorities.

## How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Use the slicers to filter by **Department**, **Cadre** (e.g., Nurse, Fellow/Doctor), or **Auditor**.
```
